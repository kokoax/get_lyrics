# GetLyrics
## Discription
  This application is getting lyrics.
  Song searching is automatic get from the top of UtaTen.com with options.

## Installation
``` shell
  $ cd get_lyrics
  $ mix deps.get
  $ mix escript.build
  $ ./get_lyrics --title "" --artist "" --composer ""
```
## Usage
``` shell
  $ get_lyrics -t "song title"
```
